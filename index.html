Bienvenido a PJSK

Project SEKAI COLORFUL STAGE! feat. Hatsune Miku (also known as "Project SEKAI"or "Proseka") is a rhythm game developed as a collaboration between SEGA, Craft Egg, Colorful Palette, and Crypton Future Media. The game was first released in Japan on September 30, 2020 for iOS and Android devices.

In addition to the original Japanese server, there are four localised versions of the game released internationally. These include the Taiwanese (Traditional Chinese) server, which released on September 30, 2021; the Global (English) server, which released on December 7, 2021; and the Korean server, which released on May 20, 2022. There is also a Simplified Chinese version of the game set to release in Mainland China on March 27th, 2025.
